dispatcher_impl library

All dispatcher and dispatcher impl classes including the global dispatcher.

If dispatcher impl classes are not needed import dispatcher.dart instead.

The global dispatcher can be hidden by appending hide dispatcher to the import statement.

import 'package:bloc_flux/dispatcher.dart' hide dispatcher;

OR

import 'package:bloc_fluc/dispatcher_impl.dart' hide dispatcher;

OR

import 'package:bloc_flux/bloc_flux.dart' hide dispatcher;

dispatcher is lazily instantiated so if it is never accessed it will not be instantiated.
