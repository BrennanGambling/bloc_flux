StateBlocState constructor

StateBlocState({BuiltMap<FieldID, StateFieldState> stateMap, BuiltList<StateFieldState> stateList })

Instaniates a StateBlocState from a collection.

blocKey will be equal to the FieldID of all of the StateFieldStates and FieldIDs. Either stateMap OR stateList should be specified but NOT BOTH, otherwise an ArgumentError will be thrown.

The specified collection (stateMap or stateList) MUST NOT be empty, otherwise an ArgumentError will be thrown.

The StateFieldState.fieldID of each of the StateFieldStates in stateList MUST BE EQUAL, otherwise an ArgumentError will be thrown.

The FieldID.blocKey keys and FieldState.fieldID.blocKey of the StateFieldState values MUST BE EQUAL, other an ArgumentError will be thrown.

Implementation

factory StateBlocState(
        {BuiltMap<FieldID, StateFieldState> stateMap,
        BuiltList<StateFieldState> stateList}) =>
    StateBlocState.fromBuilder((b) {
      final bool stateMapNull = stateMap == null;
      final bool stateListNull = stateList == null;

      _parameterChecks(stateMap, stateList);

      if (!stateMapNull) {
        b.stateMap = stateMap;
      } else if (!stateListNull) {
        final MapBuilder<FieldID, StateFieldState> mapBuilder = MapBuilder();
        mapBuilder
            .addEntries(stateList.map((fd) => MapEntry(fd.fieldID, fd)));
        b.stateMap = mapBuilder.build();
      }
    });
