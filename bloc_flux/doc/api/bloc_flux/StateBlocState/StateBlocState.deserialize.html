StateBlocState.deserialize constructor

StateBlocState.deserialize(String serialized)

Deserializes a serialized StateBlocState using the default serialization format.

serialized is first decoded using jsonDecode() and then deserialized using]] the blocFluxSerializers.

*** Differences Between Serializers blocFluxSerializers uses the "default serialization format" which has "better performance and support for more collection types.".

standardJsonSerializers uses a "simple map-based JSON" which most other systems use.

If serialized values are only used within this project and are not deserialized by other serialization libraries blocFluxSerializers should be used. Otherwise use standardJsonSerializers.

Implementation

factory StateBlocState.deserialize(String serialized) => blocFluxSerializers
    .deserializeWith(StateBlocState.serializer, jsonDecode(serialized));
