removeBloc method

void removeBloc (Bloc bloc)
override

Removes bloc from this Dispatcher.

If bloc is not registered with this Dispatcher nothing will happen.

This method will throw a StateError if this is closed. A Dispatcher is closed after its dispose method is called. To see if a Dispatcher is closed use getter closed.

Implementation

void removeBloc(Bloc bloc) {
  checkClosed();
  final String key = bloc.key;
  blocMap.remove(key);
  if (bloc is ValueBloc) {
    valueBlocMap.remove(key);
  }
  if (bloc is StateBloc) {
    stateBlocMap.remove(key);
  }
}
