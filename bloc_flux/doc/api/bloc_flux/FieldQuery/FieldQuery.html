FieldQuery constructor

FieldQuery({String blocKey, BuiltList<FieldID> fieldIDs, BuiltList<String> fieldKeys, FieldID fieldID, String fieldKey, dynamic single: false })

Instaniates a FieldQuery.

Specific Fields

To specify the Field to query specify ONE of the following fieldIDs, fieldKeys, fieldID or fieldKey.

Only specify ONE of the following parameters:

  1. fieldIDs
  2. fieldKeys
  3. fieldID
  4. fieldKey If more than one of theses parameters are specified an ArgumentError will be thrown.

If fieldKey or fieldKeys is specified blocKey MUST also be specified, otherwise an ArgumentError will be thrown.

If fieldIDs is specified the FieldID.blocKey for each element must be the same, otherwise an ArgumentError will be thrown.

All Fields

To query all of the Fields do not specify any of the ID (fieldID(s)) or Key (fieldKey(s)) parameters just the blocKey.

If none of the ID (fieldID(s)) or Key (fieldKey(s)) are specified the blocKey MUST be specified, otherwise an ArgumentError will be thrown.

One Time Request

If this query is a one time request set single to true (defaults to false).

Implementation

factory FieldQuery(
        {String blocKey,
        BuiltList<FieldID> fieldIDs,
        BuiltList<String> fieldKeys,
        FieldID fieldID,
        String fieldKey,
        single = false}) =>
    FieldQuery.fromBuilder((builder) {
      final bool fieldIDsNull = fieldIDs == null;
      final bool fieldKeysNull = fieldKeys == null;
      final bool fieldIDNull = fieldID == null;
      final bool fieldKeyNull = fieldKey == null;

      _parameterChecks(blocKey, fieldIDs, fieldKeys, fieldID, fieldKey);

      if (!fieldIDsNull) {
        builder.fieldIDs = BuiltList(fieldIDs);
      } else if (!fieldKeysNull) {
        builder.fieldIDs =
            BuiltList(fieldKeys.map((key) => FieldID(blocKey, key)));
      } else if (!fieldIDNull) {
        builder.fieldIDs = (ListBuilder()..add(fieldID)).build();
      } else if (!fieldKeyNull) {
        builder.fieldIDs =
            (ListBuilder()..add(FieldID(blocKey, fieldKey))).build();
      }

      builder
        ..single = single
        ..cancel = false;
    });
