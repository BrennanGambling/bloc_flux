expectBuilder method

  1. @override
void expectBuilder (FullType fullType)
override

See Serializers.expectBuilder().

Returns without throwing if ANY of the added Serializers do not throw.

To check if all added Serializers have a builder for fullType use allHaveBuilders and throw if true is not returned.

Implementation

@override
void expectBuilder(FullType fullType) {
  StateError stateError;
  bool found = false;
  _allSerializers.forEach((s) {
    if (found) {
      return;
    }
    try {
      s.expectBuilder(fullType);
      found = true;
      return;
    } catch (e) {
      if (e is StateError) {
        stateError = e;
      } else {
        throw e;
      }
    }
  });
  if (!found) {
    throw stateError;
  }
}
