invalidFields method

Iterable<FieldID> invalidFields (FieldQuery fieldQuery)

Returns an Iterable of FieldIDs in fieldQuery that do not have Fields in this ValueBloc with equal FieldIDs.

All FieldIDs will be returned if FieldQuery.blocKey does not equal key.

An empty list will be returned if all Fields are valid. If this Bloc is already closed calling this method will result in a StateError being thrown. For more information on when a Bloc is considered closed see Bloc.closed.
